🔌 API란?

API = 프로그램 간의 대화 방법

식당에서 주문할 때 메뉴판을 보고 주문하듯이, API는 프로그램이 다른 프로그램에게 데이터를 요청하는 규칙입니다.

REST API 메서드

GET 데이터 조회 (읽기)

POST 데이터 생성 (쓰기)

PUT 데이터 수정 (업데이트)

DELETE 데이터 삭제

🌤️ 날씨 API 체험

버튼을 누르면 가상 API 응답을 볼 수 있어요



요청/응답 구조


📤 요청 (Request)


GET /api/users/1 HTTP/1.1
Host: api.example.com
Authorization: Bearer eyJhbG...
Accept: application/json


📥 응답 (Response)


HTTP/1.1 200 OK
Content-Type: application/json

{
  "id": 1,
  "name": "홍길동",
  "email": "hong@example.com"
}
