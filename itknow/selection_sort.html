선택 정렬 시각화

선택 정렬(Selection Sort) 알고리즘의 동작 과정을 시각적으로 보여줍니다. 배열에서 최솟값을 찾아 맨 앞에 놓는 과정을 반복합니다. 시간 복잡도는 O(n²)입니다.

사용 방법

시작 버튼을 누르면 정렬 과정이 애니메이션으로 표시됩니다.

팁

버블 정렬보다 교환 횟수가 적어 약간 더 효율적입니다.
