메모리 카드 게임

뒤집힌 카드 중 같은 그림의 쌍을 찾는 기억력 게임입니다. 카드를 두 장씩 뒤집어 같은 그림이면 매칭됩니다. 모든 쌍을 찾으면 클리어입니다.

사용 방법

카드를 클릭하여 뒤집습니다. 두 장을 뒤집어 같은 그림이면 그 자리에 남고, 다르면 다시 뒤집힙니다. 적은 시도로 모든 쌍을 찾으세요.

팁

카드 위치를 체계적으로 기억하세요. 한 줄씩 순서대로 기억하면 도움이 됩니다. 이미 본 카드의 위치를 잘 기억해두세요.
