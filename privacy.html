개인정보처리방침

시행일: 2026년 3월 25일 (최종 수정)

hoit.me(이하 "본 사이트")는 무료 웹 게임, 심리테스트, 운세, 퀴즈, 도구 등 다양한 콘텐츠를 제공하는 서비스입니다.

1. 수집하는 정보

본 사이트는 다음 정보를 수집·저장할 수 있습니다:

  • 닉네임 — 사용자 직접 입력 또는 자동 부여 (예: nonamed0001)
  • 게임 점수 기록 — 리더보드 등록용
  • 심리테스트/운세 결과 — 결과 재확인용 (브라우저에만 저장)
  • 설정 정보 — 다크/라이트 모드, 업적 달성 기록
  • 페이지 조회 데이터 — 어떤 콘텐츠가 인기 있는지 분석용

이메일, 전화번호, 실명 등 개인을 식별할 수 있는 정보는 수집하지 않습니다.

2. 데이터 저장 방식

로컬 저장 (localStorage):

  • 닉네임, 게임 점수, 테스트 결과, 업적, 설정값
  • 사용자의 브라우저에만 저장되며, 브라우저 데이터 삭제 시 제거됩니다

서버 저장 (Firebase Realtime Database):

  • 리더보드용 닉네임 + 게임 점수 (일별 관리)
  • 페이지 조회 수 통계 (익명)
  • Google 클라우드(싱가포르 리전)에 저장됩니다

3. 쿠키 및 광고

본 사이트는 Google AdSense 광고를 게재하며, 이 과정에서 Google이 쿠키를 사용할 수 있습니다.

  • Google은 사용자의 관심사에 기반한 맞춤 광고를 표시할 수 있습니다
  • Google의 쿠키 사용에 대해서는 Google 개인정보처리방침을 참고하세요
  • 맞춤 광고는 Google 광고 설정에서 비활성화할 수 있습니다

4. 제3자 서비스

본 사이트는 다음 제3자 서비스를 이용합니다:

  • Google AdSense — 광고 게재
  • GitHub Pages — 웹사이트 호스팅
  • Firebase Realtime Database — 리더보드 및 분석 데이터 저장
  • Kakao JavaScript SDK — 카카오톡 공유 기능
  • Frankfurter API — 실시간 환율 정보 (환율 계산기)

각 서비스의 개인정보 처리에 대해서는 해당 서비스의 개인정보처리방침을 참고하세요.

5. SNS 공유

본 사이트는 콘텐츠 공유를 위해 카카오톡, 트위터(X), 페이스북 공유 기능을 제공합니다. 공유 버튼을 클릭하면 해당 플랫폼으로 이동하며, 공유 과정에서의 데이터 처리는 각 플랫폼의 정책을 따릅니다.

6. 아동 보호

본 사이트는 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 13세 미만의 이용자가 개인정보를 제공한 사실을 알게 된 경우 해당 정보를 즉시 삭제합니다.

7. 사용자 권리

  • 브라우저 localStorage를 삭제하여 모든 로컬 데이터를 제거할 수 있습니다
  • 리더보드에 등록된 데이터의 삭제를 요청할 수 있습니다
  • 광고 쿠키는 브라우저 설정에서 관리할 수 있습니다

8. 변경 사항

본 방침은 필요에 따라 변경될 수 있으며, 변경 시 본 페이지에 게시됩니다.

9. 문의

개인정보 관련 문의사항은 GitHub 저장소의 Issues를 통해 접수해주세요.
